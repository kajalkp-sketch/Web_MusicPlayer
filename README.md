🎵 Web Music Player

A Spotify-style web music player built using HTML, CSS, and JavaScript.

🚀 Live Demo
 https://kajalkp-sketch.github.io/Web_MusicPlayer/

📌 Features
- 🎧 Play / Pause songs
- ⏭️ Next / Previous controls
- 🔀 Shuffle functionality
- 🔊 Volume control
- 📊 Song progress bar
- 🎶 Multiple playlists (Chill & Party)
- ✅ Highlight current playing song

🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
